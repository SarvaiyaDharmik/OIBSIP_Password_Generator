# Random Password Generator

## 📌 Project Description
This is a Flask-based Random Password Generator web application developed for the Oasis Infobyte Python Programming Internship.

The application allows users to generate secure random passwords with different character combinations such as uppercase letters, lowercase letters, numbers, and symbols.

---

## 🚀 Features

- Generate secure random passwords
- User can select password length
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Input validation
- Error handling
- Simple and user-friendly interface
- Responsive UI using HTML and CSS

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
